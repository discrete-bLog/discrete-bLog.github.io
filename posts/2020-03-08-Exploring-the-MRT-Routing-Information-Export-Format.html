Exploring the MRT Routing Information Export Format

2020-03-08 - Rachel Rybarczyk

The Multi-Threaded Routing Toolkit (MRT) Routing Information Export Format is defined in RFC6396 and can be used "to export routing protocol messages, state changes, and routing information base contents".

MRT Header

The MRT extended header format is
Timestamp (4 bytes)
Epoch time stamp
0x5e651680, 1583683200, 2020-03-08 16:00 UTC
Extended Flag (1 byte)
Indicates this is the extended header format.
0x00, 00
Type (1 byte)
0x0d, 13, indicates
SubType (2 bytes)
0x0001, 1, indicates
