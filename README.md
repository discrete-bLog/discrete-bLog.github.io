# discrete bLog


#### Jekyll Commands
```
$ jekyll build      # builds `_site` dir
$ jekyll serve      # rebuild on save & runs on http://localhost:4000
$ jekyll clean      # remove `_site` dir
```

Using the justfile:
```
$ just b
$ just s
$ just c
```
